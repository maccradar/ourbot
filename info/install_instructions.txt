Install odroid
______________

- exceed memory + change host name
- network settings
- install ros + orocos
- make orocos components
- corba settings
- install casadi: you need to build from sources as the pre-built binaries does not match odroid processor!
- environment vars available
    * go to /etc/ssh/sshd_config and add option PermitUserEnvironment yes
    * call env and copy this list and put it in ~/.ssh/environment
- rt priority settings
    * go to /etc/security/limits.conf and set your rtprio limit
    odroid  hard rtprio unlimited
    odroid  hard priority unlimited
    * ulimit -r xxx
- rplidar
    * cd /etc/udev/rules.d/
    sudo touch 50-usb-serial.rules
    sudo gedit 50-usb-serial.rules

    Write in this file:
    KERNEL=="ttyUSB0", USER="$odroid$", MODE="0666"

    Reload the file calling:
    sudo /etc/init.d/udev restart
    * sudo gpasswd --add ${USER} dialout
- csm (Canonical Scan Matcher)
    sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'
    sudo apt-key adv --keyserver hkp://pool.sks-keyservers.net --recv-key 0xB01FA116
    sudo apt-get update
    sudo apt-get install ros-indigo-csm
- Eigen library
    (https://eigen.tuxfamily.org/dox/GettingStarted.html)
    * get Eigen library
    * copy Eigen folder to /usr/local/include/
